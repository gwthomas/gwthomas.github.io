Teaching

This semester (Fall 2015) I am working as a Tutor for CS 61A, which is a very large (there are ~1400 students enrolled!) and excellent class that provides an introduction to computer programming. It covers topics such as

  • how programs are evaluated
  • how to use abstraction to write maintainable software
  • common programming paradigms: imperative, object-oriented, functional, declarative
  • useful data types
and more!
